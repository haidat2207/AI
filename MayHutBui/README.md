## 🧠 Các thuật toán sử dụng

| Thuật toán | Mô tả                      |
| ---------- | -------------------------- |
| BFS        | Tìm kiếm theo chiều rộng   |
| DFS        | Tìm kiếm theo chiều sâu    |
| UCS        | Tìm kiếm chi phí thấp nhất |
| IDS        | DFS có giới hạn độ sâu     |
| Greedy     | Chọn heuristic tốt nhất    |
| A*         | Kết hợp cost + heuristic   |
